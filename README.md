# Discord-Custom-Nitro
This allows you to have custom nitro boosting messages for your server.

# Example

[Example of Custom Message](https://i.imgur.com/fdQzMsu.png)

# Setup
1. Clone this repository into a folder.
2. run the following commands.
  1. `npm init -y` or `npm init`
  2. `npm i discord.js`
3. change all the variables that need to be filled like
  - Bot Token
  - Channel ID
  - Role Name | Default: `Nitro Booster`
 4. Run `node .`
 5. profit

If something went wrong or the repo gets outdated you can always contact me on the information below!
